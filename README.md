# DeveloperFolio Bootstrap Rewrite

Static Bootstrap rewrite inspired by `saadpasta/developerFolio`.

## Tech Stack

- HTML5
- Bootstrap 5 CDN
- Bootstrap Icons CDN
- Custom CSS
- Vanilla JavaScript

## How to Run

Open `index.html` directly in your browser, or run a local server:

```bash
python -m http.server 8000
```

Then open `http://localhost:8000`.

## How to Customize

- Edit personal text in `index.html`
- Edit colors and styles in `assets/style.css`
- Edit interactions in `assets/script.js`
- Replace `assets/resume.pdf` with your real resume if needed
